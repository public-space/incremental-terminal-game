▸ TROUBLESHOOTING

Game won't start / "Permission denied"

Make sure the launcher is executable: 

chmod +x colony.sh

Python version error

colony.sh requires Python 3.8+. Check your version: 

python3 --version

If your version is too old, install a newer Python or run directly: 

python3.8 src/main.py

Display looks broken / wrong colors

Ensure your terminal supports ANSI color codes. Try:

  • Using a modern terminal (GNOME Terminal, Konsole, iTerm2)
  • Expanding your terminal window (minimum 80x24)
  • Checking terminal color settings

Save files not loading

Check that the saves/ directory exists and has proper permissions: 

mkdir -p saves
chmod 755 saves

Still having issues?

Check the GitHub Issues or create a new issue with:

  • Your OS and version
  • Python version (python3 --version)
  • Terminal emulator
  • Error message or description
